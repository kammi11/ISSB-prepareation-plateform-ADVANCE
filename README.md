# 🎖️ ISSB Master

A modern AI-powered preparation platform for candidates preparing for the Pakistan Armed Forces ISSB (Inter Services Selection Board).

The platform provides complete preparation for Intelligence Tests, Psychological Tests, GTO Tasks, Current Affairs, AI Mock Interviews, Progress Tracking, Premium Membership, and much more.

---

## 🚀 Features

### 🧠 Intelligence Tests
- Verbal Reasoning
- Non-Verbal Reasoning
- Analogies
- Number Series
- Pattern Recognition
- Timed Practice
- Instant Score

---

### 🎙️ AI Mock Interview

An AI Interview Officer asks realistic ISSB interview questions and evaluates candidate responses.

Features

- General Interview
- Military Interview
- Personal Interview
- Motivation Questions
- Leadership Questions
- AI Feedback
- Session History

---

### 🧩 Psychological Tests

Includes

- Word Association Test (WAT)

- Self Description Test

## ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/yourusername/ISSB-Master.git
```

Open

```
index.html
```

or deploy using

- GitHub Pages
- Netlify
- Vercel

---

## 🗄 Database Setup

1. Create a Supabase Project

2. Open SQL Editor

3. Run

```
supabase_premium_setup.sql
```

4. Enable Authentication

5. Add your

- Supabase URL
- Supabase Anon Key

inside the project.
